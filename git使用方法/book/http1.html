返回一个静态页面

增加静态页返回功能，用户的访问返回欢迎界面

1.新建一个文件夹Webapp

2.将静态html文件放入Wenapp文件夹当中

3.修改sockHandler,添加页面返回功能sendWebPage方法

# a.因为要使用os文件操作所以在sockHandler文件头部要引入os模块
import os

# b.把WebApp文件夹的绝对路径添加到ServerContent配置文件当中
class ServerContent:
    ip = '127.0.0.1'
    port = 9999

    head_protocal = "HTTP/1.1 "
    head_code_200 = "200 "
    head_status_OK = "OK"

    head_content_length = "Content-Length: "
    head_content_type = "Content-Type: "
    content_type_html = "text/html"

    blank_line = ""

    # 此处为添加的内容
    base_path = r'E:\py03\websocket\WebApp'

# c.添加sendWebPage方法
def sendWebPage(self, path):
    fp =  os.path.join(serverContent.base_path, path)
    with open(fp,mode='r',encoding='utf-8') as f:
        html = f.read()
        self.__sendRspAll(html)

4.修改sockHandler,startHandler方法

def startHandler(self):
    self.headHandler()
    # self.sendRsp()
    self.sendWebPage('index.html')

5.因为我们读取的文件类型格式是UTF-8，所以在返回数据时应该也是UTF-8，故修改sockHandler文件中__sendRspLine方法：

def __sendRspLine(self,data):

    data += "\r\n"
    # 此处修改编码格式为uft-8
    self.sock.send(data.encode("UTF-8"))
    return None

注意： 我们的HTML文件中因为charset=‘utf-8’，所以我们读取文件时用UTF-8的格式读取，最后我们返回数据时也要用UTF-8编码，这三处编码格式必须要统一
